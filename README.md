# 二、`vim` / `nvim` 配置

### 1. `vim`

把 `.vim` 文件夹放到当前用户的根目录下, `coc` 文件夹放到 `.config` 目录下  
然后进入 `vim` , 输入

```
:PlugInstall
或者
:PlugUpdate
```

### 2. `nvim`

把 `nvim` 文件夹放到当前用户的 `.config` 目录下, `coc` 文件夹放到 `.config` 目录下  
然后进入 `nvim`,输入

```
:PlugInstall
或者
:PlugUpdate
```
